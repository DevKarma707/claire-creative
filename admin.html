Clients

Les clients apparaissent dans le menu « Clients » du site. Ceux qui n'ont aucun projet rattaché sont affichés grisés (non cliquables). La publication se fait avec le bouton « Publier » en bas.

Importer des projets (JSON)

Connexion GitHub

L'admin publie directement sur le site via GitHub. Il faut un « fine-grained personal access token » avec l'autorisation Contents : Read and write sur le dépôt DevKarma707/claire-creative (validité conseillée : 1 an).
À créer ici : github.com/settings/personal-access-tokens/new — puis colle-le ci-dessous (il reste enregistré dans ce navigateur uniquement).

Infos

Après publication, le site se met à jour en 1 à 2 minutes ; l'admin surveille et t'indique « En ligne ✓ » quand c'est visible. Chaque publication est historisée : rien n'est jamais perdu.
