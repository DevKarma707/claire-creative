# Claire Creative — clairecreative.fr

Refonte du site de **Claire Creative**, studio de direction artistique et production événementielle à Paris, fondé par Claire Deroo.

Site statique (HTML/CSS/JS, zéro dépendance) :

- `index.html` — accueil (hero typographique, marquee, projets, services, clients, contact)
- `portfolio.html` — galerie filtrable par catégorie (deep-link `?cat=…`) + lightbox
- `about.html` — présentation FR/EN
- `assets/img/` — 142 photos réparties en 6 catégories (weddings, parties, scenography, interiors, window-display, set-design)

Signature : les yeux du logo recréés en SVG suivent le curseur et clignent.

Déployé via GitHub Pages.
